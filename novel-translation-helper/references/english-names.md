# 英文人名库

适用于中文小说角色翻译的常用英文名字，按性别和风格分类。

**重要原则**：人名翻译必须使用**完整的英文姓名**（First Name + Last Name），严禁使用拼音姓氏。

---

### 家族命名体系示例

```
中文姓氏 → 英文姓氏 → 家族成员 → 公司/组织
顾 → Gray → Stella Gray, Alexander Gray, Olivia Gray → Gray Corporation
傅 → Foster → Ethan Foster → Foster Corporation / Foster Industries
沈 → Shaw → Victoria Shaw, Daniel Shaw → Shaw Enterprises
陆 → Lewis → Emma Lewis, James Lewis → Lewis & Co.
阮 → Reyes → Chloe Reyes → Reyes Trading
```

### 公司/组织命名规则

| 类型 | 命名格式 | 示例 |
|------|----------|------|
| 大型集团 | [姓氏] Corporation / Group | Gray Corporation, Foster Group |
| 中型公司 | [姓氏] Company / Industries | Shaw Company, Foster Industries |
| 小型企业 | [姓氏] Enterprises / & Co. | Reyes Enterprises, Lewis & Co. |
| 家族住所 | [姓氏] Residence / Manor | Gray Residence, Foster Manor |
| 家族称谓 | The [姓氏]s / [姓氏] Family | The Grays, Foster Family |
