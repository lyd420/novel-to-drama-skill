# 中文小说到动漫短剧工作流

一键式完成中文小说的海外本地化、英文翻译和短剧分集改编，生成适合海外市场的动漫短剧内容。

## 工作流概览

```
┌─────────────────────────────────────────────────────────────┐
│  输入：中文小说文件 (.txt/.md)                                │
└──────────────────────────┬──────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  STEP 1: 海外本地化 (novel-overseas-localization)           │
│  - 价值观红线检查                                           │
│  - 中式文化符号本地化                                       │
│  - 剧情逻辑适配                                             │
│  - 中式表达清洗                                             │
│  输出：修改后中文版 + 分析报告                              │
└──────────────────────────┬──────────────────────────────────┘
                           │ 用户确认后继续
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  STEP 2: 英文翻译 (novel-translation-helper)                │
│  - 选择地名本地化策略（美国/英国/通用）                     │
│  - 提取术语表（人名→英文姓名）                              │
│  - 提取术语表（地名→美国地名）                              │
│  - 全文翻译                                                 │
│  输出：术语表 + 英文版小说                                  │
└──────────────────────────┬──────────────────────────────────┘
                           │ 用户确认后继续
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  STEP 3: 短剧分集 (short-drama-adapter)                     │
│  - 分析原文结构                                             │
│  - 设计短剧节奏                                             │
│  - 拆分为1-3分钟分集                                        │
│  - 生成详细分集脚本                                         │
│  输出：分集总表 + EP1-N详细脚本                             │
└──────────────────────────┬──────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  最终输出                                                   │
│  - 修改后中文版                                             │
│  - 英文版小说                                               │
│  - 分集脚本                                                 │
│  - 完整工作流报告                                           │
└─────────────────────────────────────────────────────────────┘
```

## 工作流执行步骤

### 第一步：启动工作流

用户输入：
```
请开始工作流，处理文件：[小说文件名]
地名本地化策略：[美国/英国/通用/保留真实城市]
```

或简单输入：
```
一键生成短剧：[小说文件名]
```

系统确认：
```
即将开始完整工作流，包含以下步骤：
1. 海外本地化检查与修改
2. 英文翻译（含术语表）
3. 短剧分集改编

预计耗时：[根据字数估算]
输出文件：
- [文件名]_本地化.md
- [文件名]_术语表.csv
- [文件名]_EN.txt
- [文件名]_ShortDrama.md

确认开始？(是/否)
```

### 第二步：执行STEP 1 - 海外本地化

自动调用 `novel-overseas-localization` skill：

1. **读取原文**
2. **四步检查**：
   - 海外价值观红线检查
   - 中式文化符号本地化
   - 剧情逻辑商业化适配
   - 中式表达清洗
3. **生成修改版本**
4. **输出分析报告**

**输出文件**：
- `[文件名]_本地化.md` - 修改后的中文版
- `[文件名]_分析报告.md` - 详细分析报告

**用户确认点**：
```
STEP 1 完成！

已生成本地化版本：
- 修改后文件：[文件名]_本地化.md
- 分析报告：[文件名]_分析报告.md

主要修改：
- 价值观红线：[X] 处修改
- 文化符号：[X] 处替换
- 剧情逻辑：[X] 处调整
- 表达清洗：[X] 处优化

请检查修改后的文件，确认后继续下一步？(是/否/修改意见)
```

### 第三步：执行STEP 2 - 英文翻译

自动调用 `novel-translation-helper` skill：

1. **地名策略确认**（如用户未指定，则询问）
2. **术语表提取**：
   - 人物名称 → 英文姓名（First+Last Name）
   - 地名 → 本地化地名
   - 家族姓氏 → 英文姓氏
   - 公司/组织 → 英文命名
3. **用户确认术语表**
4. **全文翻译**

**输出文件**：
- `[文件名]_术语表_[日期].csv` - 术语对照表
- `[文件名]_EN.txt` - 英文版小说

**用户确认点**：
```
STEP 2 完成！

已生成英文版本：
- 术语表：[文件名]_术语表.csv
- 英文小说：[文件名]_EN.txt

术语表预览：
[显示关键术语映射]
- 顾晓念 → Stella Gray
- 傅景行 → Ethan Foster
- 北京 → Washington, D.C.
- 傅氏集团 → Foster Corporation

地名策略：[美国本地化/英国本地化/通用]

请检查英文版本，确认后继续下一步？(是/否/修改意见)
```

### 第四步：执行STEP 3 - 短剧分集

自动调用 `short-drama-adapter` skill：

1. **分析英文版结构**
2. **识别冲突点和节奏点**
3. **规划分集**（每集1-3分钟）
4. **生成详细脚本**：
   - 分集总表
   - EP1-N 场景分解
   - Hook/Climax/Cliffhanger 设计

**输出文件**：
- `[文件名]_ShortDrama_Episodes.md` - 完整分集脚本

**用户确认点**：
```
STEP 3 完成！

已生成分集脚本：
- 分集文件：[文件名]_ShortDrama_Episodes.md

分集统计：
- 总集数：[X] 集
- 预估总时长：[X] 分钟
- 平均每集：[X] 分钟

分集分布：
- EP1-3: [标题] - [事件]
- EP4-6: [标题] - [事件]
- ...

前3集详细预览：
[显示EP1-3概要]

```

## 参数配置

### 可配置参数

| 参数 | 选项 | 默认 | 说明 |
|------|------|------|------|
| `地名策略` | 美国/英国/通用/保留 | 美国 | 地名本地化方式 |
| `分集时长` | 1-2分钟/2-3分钟/混合 | 混合 | 短剧集长 |
| `冲突密度` | 高/中/低 | 高 | 每集冲突数量 |
| `完全本地化` | 是/否 | 是 | 是否消除中国痕迹 |

### 配置文件示例

```yaml
workflow_config:
  step1_localization:
    enabled: true
    redline_check: true
    cultural_adaptation: true
    
  step2_translation:
    enabled: true
    location_strategy: "usa_full"  # usa_full/uk/full/generic/keep
    name_translation: "full_english"  # first+last name
    
  step3_drama_adaptation:
    enabled: true
    episode_duration: "mixed"  # 1-2min/2-3min/mixed
    pacing: "high_conflict"  # high/medium/low
    
output:
  generate_report: true
  save_intermediate: true
```

## 中断与恢复

### 中断工作流

用户可随时中断：
```
暂停工作流
```

系统保存进度，下次可从断点继续。

### 恢复工作流

```
恢复工作流，从STEP [X]继续
```

### 单步执行

也可只执行特定步骤：
```
只执行STEP 2，文件：[文件名]_本地化.md
```

## 依赖技能

本工作流依赖以下技能：

1. **novel-overseas-localization**
   - 路径：`novel-overseas-localization/SKILL.md`
   - 作用：STEP 1 - 海外本地化

2. **novel-translation-helper**
   - 路径：`novel-translation-helper/SKILL.md`
   - 作用：STEP 2 - 英文翻译

3. **short-drama-adapter**
   - 路径：`short-drama-adapter/SKILL.md`
   - 作用：STEP 3 - 短剧分集

## 故障排除

### 常见问题

**Q: STEP 1修改后需要大量人工调整？**  
A: 可以暂停工作流，修改 `[文件名]_本地化.md` 后，使用 `恢复工作流` 继续。

**Q: 术语表翻译不满意？**  
A: 在STEP 2的用户确认点，回复修改意见，如："修改：顾晓念 → Sophia Gray"

**Q: 分集节奏不符合预期？**  
A: 可以重新执行STEP 3，调整 `分集时长` 和 `冲突密度` 参数。

**Q: 处理大文件时内存不足？**  
A: 可以分段处理，或增加 `batch_size` 配置。

## 执行建议

- **大文件处理**：超过10万字的小说建议分卷处理
- **人工审核**：每个STEP的用户确认点建议仔细审核
- **迭代优化**：可多次运行工作流，逐步优化输出质量
- **版本管理**：建议保存每次运行的输出，便于对比
