# 快速开始指南

novel-to-shortdrama-workflow 的快速使用指南。

## 一键启动

### 最简命令
```
一键生成短剧：我的小说.txt
```

系统将使用默认配置：
- 美国完全本地化
- 混合时长（1-3分钟）
- 高冲突节奏

### 指定策略
```
开始工作流，文件：我的小说.txt，地名策略：美国本地化
```

## 故障排除

### 文件无法读取
```
错误：无法读取文件 "小说.txt"
解决：请检查文件路径是否正确
```

### 字数超限
```
警告：文件字数超过10万字，建议分卷处理
选项：1.自动分卷 2.手动分割 3.继续处理
```

### 术语表冲突
```
警告：检测到姓氏 "Gray" 已被其他小说使用
建议：是否使用替代姓氏 "Grey" 或 "Grayson"？
```

### 步骤失败
```
[STEP 2] 翻译失败
原因：术语表包含无法翻译的专有名词
解决：请检查术语表并手动指定翻译
```

## 最佳实践

### 文件准备
- 使用UTF-8编码的txt或md文件
- 章节标题使用"第X章"或"Chapter X"
- 段落间留空行，便于解析

### 工作流程
1. 先执行STEP 1，仔细检查本地化修改
2. 确认术语表，特别是人物姓名是否满意
3. 检查英文版的流畅度
4. 最后生成分集脚本

### 质量检查
- 每步完成后都建议人工审核
- 重点关注：价值观修改、人名翻译、分集节奏
- 保留每次运行的输出，便于对比
