# 短剧结构指南

动漫短剧（1-3分钟）的标准结构和节奏控制。

## 短剧基本结构

### 1分钟短剧结构

```
【0:00-0:05】Hook（5秒）
- 最冲突的画面
- 立即抓住注意力

【0:05-0:20】Setup（15秒）
- 快速交代背景
- 人物关系

【0:20-0:45】Conflict（25秒）
- 主要冲突展开
- 对话交锋

【0:45-1:00】Climax + Cliffhanger（15秒）
- 高潮或反转
- 悬念结尾
```

### 2分钟短剧结构（标准）

```
【0:00-0:05】Hook（5秒）
- 强视觉冲击
- 或强情绪冲击

【0:05-0:25】Setup（20秒）
- 背景交代
- 人物登场

【0:25-1:00】Rising Action（35秒）
- 冲突升级
- 阻碍出现

【1:00-1:30】Climax（30秒）
- 高潮时刻
- 打脸/反转

【1:30-2:00】Cliffhanger（30秒）
- 悬念揭晓或新悬念
- 引导下一集
```

### 3分钟短剧结构（复杂情节）

```
【0:00-0:05】Hook（5秒）
【0:05-0:30】Setup（25秒）
【0:30-1:15】Rising Action（45秒）
【1:15-1:45】Climax（30秒）
【1:45-2:15】Falling Action（30秒）
【2:15-3:00】Cliffhanger（45秒）
```

## 短剧节奏控制

### 节奏曲线

```
情绪强度
    │
    │        ╱‾‾‾‾＼
    │       ╱      ＼____
    │      ╱              ＼___
    │  ___╱                    ＼___
    │_/                              ＼___
    └──────────────────────────────────────→ 时间
     Hook  Setup  Conflict  Climax  Cliffhanger
```

### 关键节奏点

| 时间点 | 节奏要求 | 效果 |
|--------|----------|------|
| 0-5秒 | 必须Hook | 留住观众 |
| 30秒 | 小高潮 | 保持兴趣 |
| 60秒 | 大高潮 | 满足期待 |
| 结尾 | 强悬念 | 引导续看 |

## Hook类型

### 1. 冲突型Hook（最常用）
- 人物激烈争吵
- 肢体冲突画面
- 东西摔碎/泼洒
- 羞辱性场景

**示例**：
```
[画面：红酒泼在女主脸上]
CHLOE: "You worthless nobody!"
```

### 2. 悬念型Hook
- 神秘人物出现
- 惊人言论
- 意外发现
- 身份质疑

**示例**：
```
[画面：黑衣人递出一份文件]
MAN: "She's not who you think she is."
```

### 3. 情感型Hook
- 哭泣特写
- 绝望表情
- 背叛瞬间
- 离别场景

**示例**：
```
[画面：女主泪流满面]
STELLA: "I trusted you..."
```

### 4. 视觉型Hook
- 豪华场景
- 惊艳造型
- 特效画面
- 动作场景

**示例**：
```
[画面：豪车驶入豪宅，女主华丽登场]
NARRATOR: "The queen is back."
```

## Cliffhanger类型

### 1. 身份揭露型
```
ANTAGONIST: "You think you're so special? You're just a—"
[黑屏]
TO BE CONTINUED
```

### 2. 危机中断型
```
[反派举起刀]
[突然一只手抓住反派手腕]
[画面定格在神秘人脸上]
TO BE CONTINUED
```

### 3. 意外反转型
```
STELLA: "I'm leaving you."
ETHAN: "You can't! You're—"
[手机响起，打断对话]
TO BE CONTINUED
```

### 4. 新威胁型
```
[主角刚解决一个问题]
[手机收到威胁短信]
[镜头拉近："I know your secret"]
TO BE CONTINUED
```

## 短剧改编技巧

### 从小说到短剧的转换

**原文（小说）**：
```
Stella walked into the room. The lights were dim. She felt nervous.
She thought about what had happened yesterday. It had been a difficult day.
Finally, she saw Ethan sitting at the table.
```

**改编（短剧）**：
```
[SCENE 1 - 0:00-0:05]
HOOK: Close-up of Stella's determined face
STELLA: "Time to end this."

[SCENE 2 - 0:05-0:10]
CUT TO: Ethan smirking at the table
ETHAN: "You came."

[直接进入冲突]
```

### 删减原则

**删减内容**：
- ❌ 细腻的心理描写 → 用表情/动作代替
- ❌ 环境氛围铺垫 → 用视觉直接呈现
- ❌ 背景信息交代 → 对话中快速带过
- ❌ 重复的情节 → 只保留最激烈版本

**保留内容**：
- ✅ 所有冲突场面
- ✅ 身份揭露时刻
- ✅ 打脸/复仇场景
- ✅ 关键对话

### 合并技巧

**多场景合并示例**：

原文：
- Scene 1: 公司会议争吵
- Scene 2: 走廊对话
- Scene 3: 办公室对峙

改编后：
```
[一集内完成]
Scene 1: 会议争吵 → 直接升级
Scene 2: 走廊对话内容用一句话交代
Scene 3: 办公室对峙（高潮）
```

## 分集类型

### 1. 冲突集（最常见）
- 时长：1.5-2分钟
- 结构：Setup → Conflict → Climax
- 特点：一集解决一个小冲突

### 2. 揭秘集
- 时长：2-3分钟
- 结构：悬念 → 调查 → 揭露
- 特点：重大身份/秘密揭露

### 3. 打脸集
- 时长：1-2分钟
- 结构：羞辱 → 反转 → 复仇
- 特点：观众最期待的类型

### 4. 过渡集
- 时长：1-1.5分钟
- 结构：简述 → 转向新冲突
- 特点：承上启下，快速过渡

### 5. 情感集
- 时长：2-3分钟
- 结构：误会 → 解释 → 和解/决裂
- 特点：感情线推进

## 爆款短剧公式

### 公式1：羞辱-反击型
```
EP1: 女主被羞辱 → 隐忍
EP2: 继续被欺负 → 酝酿反击
EP3: 霸气反击 → 打脸成功 + Cliffhanger
```

### 公式2：身份揭秘型
```
EP1: 被瞧不起 → "她只是个..."
EP2: 神秘迹象出现 → 怀疑
EP3: 身份揭露 → 众人震惊 + 新悬念
```

### 公式3：复仇型
```
EP1: 被害 → 发誓复仇
EP2: 以新身份归来 → 布局
EP3: 开始复仇 → 第一个目标倒下
```

### 公式4：真假千金型
```
EP1: 真千金被假千金欺负
EP2: 真相初现端倪
EP3: 身份对质 → 揭穿假千金 + 反转
```
