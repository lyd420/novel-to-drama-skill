# 中式文化符号本地化指南

处理海外读者难以理解或易误解的中式文化元素。

## 一、语言与文学元素

### 成语与典故
| 类型 | 示例 | 处理建议 |
|------|------|----------|
| 常见成语 | 画龙点睛、守株待兔 | 替换为直白描述 |
| 生僻典故 | 叶公好龙、东施效颦 | 删除或改为通用比喻 |
| 诗词引用 | 古诗词、对联 | 翻译含义或直接删除 |

**示例**：
- 原文："他真是守株待兔"
- 改写："他只是被动等待机会，而不是主动出击"

### 网络梗与流行语
| 类型 | 示例 | 处理建议 |
|------|------|----------|
| 中文网络梗 | 666、yyds、破防 | 删除或替换为情绪描述 |
| 古风梗 | 臣妾、奴家、洒家 | 改为标准现代用语 |
| 二次元用语 | 腹黑、傲娇、吐槽 | 用性格描述替代 |

## 二、社会关系与礼仪

### 亲属称谓
**问题**：中式亲属称谓过于复杂，海外读者难以建立人物关系图谱

| 中文称谓 | 含义 | 建议处理 |
|----------|------|----------|
| 堂哥/堂弟/表哥/表弟 | 父系/母系男性同辈 | 统一为 "cousin" 或直接叫名字 |
| 伯父/叔叔/舅舅/姑父 | 父母辈男性 | 简化为 "uncle" 或直接名字 |
| 七大姑八大姨 | 众多女性亲戚 | 简化为 "relatives" |

### 人情面子文化
**问题**："给面子"、"欠人情"等概念在海外文化中不存在

| 中式表达 | 含义 | 改写建议 |
|----------|------|----------|
| 给面子 | 顾及尊严 | 改为 "show respect" / "save his dignity" |
| 欠人情 | 欠下人情债 | 改为 "owe a favor" / "be indebted" |
| 人情世故 | 社交规则 | 改为 "social etiquette" / "unwritten rules" |
| 来而不往非礼也 | 礼尚往来 | 直接删除或改为 "reciprocate" |

### 辈分压制
**问题**：长幼尊卑过于严格，不符合欧美平等价值观

**处理**：
- 弱化年龄/辈分带来的权威
- 强调能力和人格平等
- 长辈改为建议而非命令

## 三、节日与习俗

### 传统节日
| 节日 | 核心元素 | 处理建议 |
|------|----------|----------|
| 春节 | 红包、团圆、鞭炮 | 简化为 "New Year celebration" |
| 中秋 | 月饼、团圆 | 可保留，加简单解释 |
| 清明 | 扫墓、祭祖 | 改为 "memorial day" |
| 七夕 | 牛郎织女 | 改为 "Chinese Valentine's Day" 或直接删除典故 |

### 中式礼仪
| 习俗 | 含义 | 处理建议 |
|------|------|----------|
| 作揖 | 拱手礼 | 改为握手、点头或直接删除动作描述 |
| 跪拜 | 下跪 | 改为鞠躬或降低频率 |
| 敬茶 | 表示尊敬 | 改为 "offer tea" 无需过多解释 |
| 红包 | 礼金 | 改为 "gift money" / "envelope with money" |

## 四、饮食与生活方式

### 中式食物
**保留原则**：常见食物保留中文名加简单解释，过于复杂的删除

| 食物 | 处理建议 |
|------|----------|
| 饺子/面条/米饭 | 保留，海外读者熟悉 |
| 火锅/烤鸭/小笼包 | 保留，加简单描述 |
| 佛跳墙/满汉全席 | 简化为 "feast" / "banquet" |
| 药膳/特殊食材 | 删除或改为普通食物 |

### 生活方式元素
| 元素 | 处理建议 |
|------|----------|
| 四合院/胡同 | 改为 "courtyard house" / "alley" |
| 麻将/象棋 | 简化为 "board game" 或直接删除 |
| 晨练/太极 | 改为 "morning exercise" |
| 茶文化 | 可保留，适度简化 |

## 五、传统思想与信仰

### 中医元素
**处理原则**：非核心元素删除，核心元素简化

| 元素 | 处理建议 |
|------|----------|
| 把脉 | 改为 "check pulse" 或删除 |
| 针灸 | 可保留，简单提及 |
| 中药/草药 | 改为 "herbal medicine" |
| 阴阳五行 | 非核心则删除 |
| 气血/经络 | 删除或改为现代医学概念 |

### 风水与命理
| 元素 | 处理建议 |
|------|----------|
| 风水 | 改为 "superstition about layout" 或删除 |
| 算命/八字 | 改为 "fortune telling" 或删除 |
| 生肖 | 简化为 "zodiac sign" |
| 吉日/凶日 | 改为 "lucky day" 或直接删除 |

### 宗教与信仰符号
| 元素 | 处理建议 |
|------|----------|
| 佛像/佛珠 | 非核心则删除 |
| 护身符/平安符 | 改为 "lucky charm" |
| 寺庙祈福 | 简化为 "pray at temple" |
| 烧香拜佛 | 删除或极度简化 |

## 六、剧情逻辑适配

### 职场与商业逻辑
| 中式逻辑 | 海外障碍 | 改写方向 |
|----------|----------|----------|
| 靠关系上位 | 与能力主义冲突 | 改为展现真实能力，或关系作为辅助 |
| 酒桌文化决定业务 | 海外无此文化 | 改为专业能力和方案质量 |
| 论资排辈 | 与平等观念冲突 | 改为能力导向 |
| 领导绝对权威 | 与扁平化管理冲突 | 改为民主讨论、集体决策 |

### 家族与继承
| 中式逻辑 | 海外障碍 | 改写方向 |
|----------|----------|----------|
| 长子继承 | 海外无此传统 | 改为能力继承或遗嘱指定 |
| 传男不传女 | 性别歧视 | 删除性别限制 |
| 家族荣誉高于个人 | 个人主义冲突 | 平衡家族与个人选择 |
| 父母之命媒妁之言 | 与恋爱自由冲突 | 改为自主恋爱 |

### 人际关系处理
| 中式逻辑 | 海外障碍 | 改写方向 |
|----------|----------|----------|
| 隐忍不发、委曲求全 | 与直接沟通文化冲突 | 改为直接表达不满 |
| 为面子维持表面和谐 | 海外更重视真诚 | 改为坦诚沟通 |
| 牺牲小我成全大我 | 个人价值被忽视 | 强调个人价值与集体平衡 |
| 以德报怨 | 过于理想化 | 改为合理边界和自卫 |

## 处理优先级

1. **必须处理**：影响剧情理解的文化元素
2. **建议处理**：可能造成阅读障碍的元素
3. **可选处理**：增添风味但不影响理解的元素
